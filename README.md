## Optimizer Demo

This repository includes some demo optimizers.

Note: The project refers to [动手学深度学习](https://zh.d2l.ai/)

Datasets:

* `dataset1`: MNIST

Models:

* `model1`: MLP

Optimizers:

* `optimizer1`: SGD
* `optimizer2`: SGDM
* `optimizer3`: AdaGrad
* `optimizer4`: RMSProp
* `optimizer5`: Adam

### Unit Test

```shell
PYTHONPATH=. python loaders/loader.py
```

* for module
